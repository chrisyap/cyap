# cyap — Chris Yap · Portfolio

Personal portfolio and landing page for [Chris Yap](https://github.com/chrisyap) — Designer × Developer.

Built as a single, self-contained HTML page with embedded CSS and JS. No build step, no framework overhead.

## Design

- **Palette:** Dark void (#080b0f) with lime (#b8ff57), coral (#ff6b47), and blue (#4d9fff) accents
- **Typography:** Fraunces (display), Space Grotesk (body), Space Mono (mono)
- **Features:** SVG noise overlay, animated grid background, scroll-triggered reveal animations, nav hide/show on scroll, active section tracking

## Sections

- Hero with terminal badge and stat cards
- About / profile
- Skills (3-column grouped chips)
- Experience timeline
- Selected work / projects
- Education
- Contact with links

This repo is deployed via GitHub Pages from the `main` branch.
